Getting started on Mac OS X

written by Mitja Kleider on 2014-10-30

Here are some notes I took using OSX for the first time, coming from the Linux world.

  • Use Ansible (or a similar tool you prefer) to manage your software state and reproduce it in the future. When I set up the system, I did not, but this would have been really helpful.
  • Install open-source software using homebrew. This will be your package manager.
  • Install proprietary software using homebrew-cask, a plugin for homebrew.
  • Disable saving to cloud if you care about private data.
  • For backups with time machine, you can build your own time capsule using a Raspberry Pi.
