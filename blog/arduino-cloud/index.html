Creating Arduino sketches in the cloud

written by Mitja Kleider on 2016-06-03

In the past, I wrote about using Arduino without the official IDE. It is also possible to write sketches in the cloud, which requires a browser plugin to upload sketches to your board.

There are two interesting solutions:

  1. codebender has been around for some time and supports many different boards.
  2. Arduino CREATE, including the official web editor, is still in beta phase.
