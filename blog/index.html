Spacemacs

written by Mitja Kleider on 2016-06-04

Spacemacs combines the power of emacs with vim keybindings and sensible default configuration.

I have also tried the pure variants as well as other popular editors like Atom and Sublime. I was already used to vim keybindings (which are probably better for your tendons), but I really wanted to use org mode. Finally I ended up with Spacemacs.

Creating Arduino sketches in the cloud

written by Mitja Kleider on 2016-06-03

In the past, I wrote about using Arduino without the official IDE. It is also possible to write sketches in the cloud, which requires a browser plugin to upload sketches to your board.

There are two interesting solutions:

  1. codebender has been around for some time and supports many different boards.
  2. Arduino CREATE, including the official web editor, is still in beta phase.

The open attitude

written by Mitja Kleider on 2016-05-31

Knowledge is what advances humanity. It is a valuable resource which can be multiplied with reasonable effort. Shared knowledge can empower the unprivileged, and contribute to a better world in many other ways. If possible, knowledge should be open.

This applies to software as well. I prefer open-source software whenever there is a suitable option.

Data collected by public institutions (excluding personal information) should be open data. Publishing data funded by taxpayers benefits the general public as well as local industry. A great example of open geographical data is OpenStreetMap. The level of detail in remote regions still surprises me.

Scientific knowledge is often hidden behind publishers' paywalls for the non-academic public or less resourceful research institutions. The Open Access movement is addressing that issue. Luckily, many research institutions and research funding entitys have adopted an open access policy.

Current copyright law, intended to protect the interests of artists, but mostly benefitting big players in the media industry, turns out to restrain creativity. The free culture movement provides licenses to work around traditional copyright while protecting the artist's interests. These licenses can be applied to writing, music, images, and more.

Version control using git

written by Mitja Kleider on 2016-05-31

While writing, it is usually neccessary to organize multiple revisions of the files. Manual organization is quite inefficient, there are specialized tools that have features like reverting changes, merging changes made by several people, showing differences between revisions and many more.

The most popular tool for revision control is git. It works with any text-based files: text, source code, spreadsheet data, and others. The git website offers great documentation and tutorials.

If you need to share your files with others or want to make them available on a server to access them from several devices, there are many options. A few are sketched here:

  • The most popular git hosting provider is Github. Many open source projects can be found here, and it can also be used for private files.
  • An open-source alternative with similar feature set is Gitlab, available as hosted or self-hosted variants.
  • You can host your own server running with an authorization solution like gitolite.
  • If you work with a team and want a review process, you can use Gerrit for access control as well.
